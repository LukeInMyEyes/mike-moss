# Google Form — New Property Enquiry

Form linked directly to the Master Google Sheet (`Leads` tab).

## Fields

### Section 1 — Lead Info

| Field | Type | Options / Notes |
|-------|------|-----------------|
| Lead Originator | Dropdown | Nico / Mike / Dave |
| Lead Source | Dropdown | Prop24 / ToLet Board / Return Business / LinkedIn / Referral / Walk-in / Other |
| Area Assignment | Dropdown | North / South / West |
| Assigned Broker | Dropdown | Nico / Mike / Dave |
| Commission Split | Short text | e.g. 50/50 Mike/Nico — leave blank if standard |

### Section 2 — Company / Tenant

| Field | Type | Notes |
|-------|------|-------|
| Company Enquiring | Short text | e.g. Unitrans |
| Contact Person | Short text | |
| Contact Number | Short text | |
| Contact Email | Email | |

### Section 3 — Requirement

| Field | Type | Options / Notes |
|-------|------|-----------------|
| Property Type | Dropdown | Warehouse / Office / Retail / Land / Other |
| Size Required (m²) | Number | |
| Power Required (Amps) | Number | Leave blank if unknown |
| Budget (R/m²/month) | Number | Leave blank if unknown |
| Location Preference | Short text | Suburb or area e.g. Pinetown, Briardene |
| Timeline | Dropdown | ASAP / 1–3 months / 3–6 months / 6–12 months / Planning only |

### Section 4 — Notes

| Field | Type | Notes |
|-------|------|-------|
| Additional Notes | Long text | Any other relevant info |
